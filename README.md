# Catálogo de Jogos Digitais

Projeto web para consulta de jogos digitais, desenvolvido com HTML, CSS e JavaScript puro. Futuramente será integrado React.
Para a API escolhi a IGDB da twitch, pois é a mais completa
Para a Framework estou pensando na utilização da Material UI(MUI)
Talvez o uso de Tailwind

## Estrutura Inicial

```
Catalogo-Jogos/
├── index.html
├── style.css
└── app.js
```

## Como executar
De começo apenas abrir o `index.html` no navegador. Não é necessário instalar dependências.
